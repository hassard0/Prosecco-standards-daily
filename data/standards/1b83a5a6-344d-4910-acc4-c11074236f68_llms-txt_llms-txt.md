# llms.txt (llms.txt)

**Status:** Emerging | **Organization:** Answer.AI

## Description
llms.txt is a standardized markdown file that gives LLMs concise, expert-level context and pointers to detailed docs, avoiding noisy HTML. It is human- and machine-readable in a precise format suitable for deterministic parsing (parsers/regex). Sites are encouraged to offer clean markdown versions of pages at the same URL with .md appended (or index.html.md for path URLs) to support reliable ingestion.

## Latest Summary
The `llms.txt` standard is a proposal for a structural markdown-based file format designed to help Large Language Models (LLMs) quickly understand the context, rules, and documentation of a project. Hosted by Answer.AI, the standard suggests placing an `llms.txt` file at the root of a website or project to provide a "handbook" for AI tools.

The format is strictly defined yet simple:
- **Header**: A single H1 tag for the project name.
- **Summary**: An optional blockquote providing a high-level overview.
- **Project Information**: A free-text area (often including bulleted lists) for specific rules, mental models, or constraints.
- **Sections**: H2 headers (like `## Docs` or `## Examples`) containing markdown lists of links to full documentation files.
- **Optional/Expanded Context**: A special `## Optional` section for secondary information that shouldn't always be included in the primary context window.

By providing this file, developers enable LLMs and AI agents (like GitHub Copilot, Cursor, or ChatGPT) to discover and ingest high-quality, relevant documentation rather than relying on outdated training data or expensive broad-crawling of a site. The standard is supported by a Python library (`llms-txt`) that can parse these files and convert the linked resources into a single consolidated XML context document suitable for prompt injection.

## What's New
Recent updates to the `llms.txt` ecosystem focus on cross-platform parsing and practical tooling. A **JavaScript implementation** has been introduced, allowing for client-side parsing of `llms.txt` files into JSON objects. The **Python library** (version reachable via `pip install llms-txt`) now includes a CLI that can automatically fetch all links defined in a remote `llms.txt` and bundle them into an XML format optimized for LLM context windows. Additionally, established frameworks like **FastHTML** and **MonsterUI** have officially adopted the standard, providing live `llms.txt` files that document their specific APIs and "rules to remember" for AI-assisted coding.
