# Agent Commerce Kit (ACK)

**Status:** Emerging | **Organization:** Catena Labs

## Description
Agent Commerce Kit (ACK) is an open-source toolkit that enables AI agents to securely participate in commerce through two protocols: ACK-ID for verifiable agent identity using W3C DIDs and VCs, and ACK-Pay for standardized agent-native payments across diverse settlement rails.

## Latest Summary
The **Agent Commerce Kit (ACK)** is an open-source framework designed to provide AI-native financial infrastructure for the emerging agent economy. Developed by **Catena Labs**, the kit addresses the unique challenges of "agentic commerce," where autonomous AI agents must handle payments, identity verification, and contractual obligations without human intervention. 

The standard focuses on three core pillars:
1.  **AI-Native Payments:** Infrastructure that allows agents to hold balances, execute transactions, and manage budgets autonomously.
2.  **Autonomous Identity:** Specialized identity frameworks for agents to verify their "human-in-the-loop" status or organizational affiliation.
3.  **Commerce Protocols:** Standardized ways for agents to negotiate prices, manage subscriptions, and interact with existing e-commerce storefronts.

The kit is primarily distributed via GitHub and is designed to be developer-first, offering SDKs and APIs that allow builders to integrate commerce capabilities into AI agents regardless of the underlying LLM (Large Language Model) or agent framework. By providing these "bottleneck" solutions—specifically around trust and financial execution—ACK aims to transition AI from mere assistants to active economic participants. The project is backed by a growing community on Discord (hosted by Catena) and is documented extensively for enterprise and independent developer use cases. Currently, the project is in an active development phase, focusing on expanding its integration library and refining its identity protocols.
