# ATEP: Agent Trust and Execution Passport (ATEP)

**Status:** Draft | **Organization:** IETF

## Description
An open standard for representing an AI agent's verifiable track record of work across marketplaces and platforms. It defines a portable, machine-readable credential format encoding execution history, capability domains, and trust tiers computed from append-only logs.

## Latest Summary
The Agent Trust and Execution Passport (ATEP) is a proposed open standard designed to create a verifiable, portable track record for AI agents. As AI agents increasingly operate across diverse marketplaces, ATEP addresses the lack of a universal reputation system, preventing agents from having to "start from zero" when moving between platforms. The standard defines a machine-readable credential format (JSON) that encodes an agent's execution history, including total and successful sessions, success rates, and specialized capability domains.

Key components of the ATEP framework include:
- **Trust Tiers:** A four-level hierarchy (Unverified, Basic, Verified, Trusted) that defines an agent's reliability. Promotion through these tiers is monotonic and based on verifiable performance thresholds, such as completing a minimum number of sessions or undergoing manual review.
- **Badge System:** A mechanism for agents to earn specific certifications or achievements based on their work history, providing a granular view of their expertise.
- **Privacy Gating:** The standard specifies two versions of the passport: a **Full Passport** (Private) containing sensitive identifiers and cryptographic keys, and a **Public Passport** (Marketplace-safe) intended for public display without exposing internal metadata.
- **Verifiability:** Passports are computed from append-only execution logs, ensuring that statistics cannot be manually inflated or fabricated.

ATEP is designed to integrate with other emerging agent standards, such as VCAP (Verifiable Cost Acceptance Protocol) for trust-gated escrow and AIVS for session verification. It serves as a foundational trust layer for agent-to-agent (A2A) commerce, allowing potential hirers to make informed decisions based on proven performance.

## What's New
The ATEP standard has been formally introduced via its first Internet-Draft (`draft-stone-atep-00`) in March 2026. This initial release establishes the JSON schema for both private and public passports, defines the four-tier trust model (Unverified to Trusted), and outlines the promotion logic based on session counts and cryptographic identity provisioning. The specification also introduces a specialized badge system and establishes clear relationships with related protocols like VCAP and Agent Protocol. Notable recent focus is on ensuring "reputation portability," allowing agents to carry their earned trust scores across different marketplaces.
