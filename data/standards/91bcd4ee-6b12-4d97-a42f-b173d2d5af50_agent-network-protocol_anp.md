# Agent Network Protocol (ANP)

**Status:** Emerging | **Organization:** Agent Network Protocol

## Description
An open-source agent communication protocol designed for secure, decentralized communication between AI agents, often described as the HTTP of the Agentic Web. It provides a multi-layer architecture including identity, meta-protocol, and application layers for agent interconnectivity.

## Latest Summary
The Agent Network Protocol (ANP) is an open-source communication framework designed to serve as the "HTTP of the Agentic Web." It enables secure, decentralized interactions between AI agents through a structured multi-layer architecture. ANP addresses the fragmentation in the AI agent ecosystem by providing a standardized way for agents to describe their capabilities, discover one another, and conduct peer-to-peer transactions. 

Key technical components include the **DID:WBA** (Decentralized Identifier: Web-Based Agent) method for identity management and a series of meta-protocols for communication. The protocol stack is divided into specific functional areas:
- **Agent Description Protocol (ANP-07):** Standardizes how agents publish their metadata and functional capabilities.
- **Agent Discovery Protocol (ANP-08):** Defines mechanisms for agents to find each other in a decentralized network.
- **Agent Communication Meta-Protocol (ANP-06):** Establishes the envelope and routing logic for messages.
- **Peer-to-Peer Agent Transaction Specification (ANP-09):** Facilitates secure value exchange or task-based contracting between agents.

The project maintains a strong open-source presence on GitHub and fosters a developer community through Discord, aiming to build a cross-platform, vendor-neutral infrastructure for the next generation of AI autonomy.

## What's New
The ANP team has recently released several critical specification drafts to complete the protocol stack. Notable additions include **ANP-07 (Agent Description Protocol)** and **ANP-08 (Agent Discovery Protocol)**, which transition the project from simple messaging to a full-service discovery network. The team has also published a comparative analysis between **ANP and MCP (Model Context Protocol)**, positioning ANP as a more decentralized alternative for broad agent-to-agent networking. Recent community activity shows a focus on the **DID:WBA Method Specification**, establishing a web-native approach to agent identity.
